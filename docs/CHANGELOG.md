# AccessOps Toolkit v1.0.0

Initial public portfolio release.

## Features

- IAM access auditing
- Access provisioning workflows
- Access removal workflows
- Dynamic system discovery
- Rich CLI interface
- Tkinter GUI
- FastAPI REST API
- Swagger/OpenAPI documentation
- Mock mode support
- Dry-run execution mode
- Platform metrics endpoint
- 28 modular system connectors

## REST API

Endpoints included:

- GET /api/systems
- GET /api/stats
- GET /api/users/{username}/access
- POST /api/users/{username}/grant/{system}
- POST /api/users/{username}/remove/{system}

## Documentation

- Architecture diagrams
- Workflow examples
- API screenshots
- Configuration guides

## Purpose

AccessOps Toolkit demonstrates IAM automation, operational tooling, FastAPI development, workflow orchestration, and platform design concepts in a portfolio-safe environment.
