Since multiple scripts require authentication credentials (LDAP, DB access, system APIs), we need to:
✅ Create Azure Key Vault secrets for:

SOC system credentials (soc-db-user, soc-db-pass, etc.)
API Keys and service credentials for:
AWS (aws_access_key, aws_secret_key)
PagerDuty (pagerduty_api_key)
Grafana (grafana_api_token)
JIRA (jira_api_token)
Auth0 (auth0_client_id, auth0_client_secret)
Snowflake (snowflake_user, snowflake_password)
And any other systems that require authentication

✅ Update all scripts to retrieve credentials from Key Vault instead of hardcoding.
✅ Test credential retrieval and error handling for missing or invalid secrets.
