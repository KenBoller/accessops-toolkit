SOC Access Management Tool - Project Summary & Progress Update
🔹 Project Overview

The SOC Access Management Tool is designed to streamline user access management across multiple systems within the Marchex SOC. It provides a unified interface for SOC analysts to check, grant, and remove access across various platforms while maintaining logging, RT ticket updates, and mock mode support.

🔹 Core Features

✔ Check User Access – Runs system-specific scripts to verify user permissions.
✔ Grant User Access – Executes system-specific scripts to provision access.
✔ Remove User Access – Ensures deprovisioning using system removal scripts.
✔ RT Ticket Logging – Automatically updates RT tickets with access changes.
✔ GUI & CLI Modes – Users can choose between an intuitive GUI or traditional CLI execution.
✔ Mock Mode – Safe testing mode for changes before applying them.
✔ Modular System Support – Dynamically loads system scripts for each action.
✔ Standardized Logging – Maintains logs in Logs/SOC_Access_Log.txt.
